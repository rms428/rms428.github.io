# Personal Academic Website — Cella Schnabel

## Project Overview
A minimal personal website for Cella Schnabel, PhD student at the University of California, Santa Barbara.
Will be hosted at **rms428.github.io** via GitHub Pages.

## Tech Stack
- Plain HTML + CSS only — no frameworks, no build tools, no JavaScript
- One page for now (just the landing/home page)
- Fonts via Google Fonts (Inter)

## Style Guidelines
- White background (`#ffffff`)
- Very minimal and clean — generous whitespace, no clutter
- Font: Inter (clean, modern, readable)
- Text color: near-black (`#111111`) with muted secondary text (`#555`)
- No heavy borders, shadows, gradients, or decorative elements
- Mobile-friendly

## File Structure
```
website/

├── style.css         ← all styles live here, no inline styles
```

## Home Page Content
- **Name:** Cella Schnabel (displayed as large heading)
- **Photo:** profile image (`images/profile.jpg`) — circular crop
- **Bio:** Short paragraph(s) about being a PhD student at UC Santa Barbara

## GitHub Pages Setup
- Repo name must be exactly: `rms428.github.io`
- Keep repo **private** until ready to go live
- To publish: change repo visibility to public in GitHub settings
- Branch: `main`

## Conventions
- All CSS in `style.css` — no inline styles
- Semantic HTML (use `<main>`, `<h1>`, `<p>`, etc.)
- Keep it simple — resist adding things that weren't asked for
