Examining Collective Action for Rangeland Conservation

This project evaluates how community-led grazing management in northeastern Kenya improves rangeland health. We spend a lot of time thinking carefully about how we define rangeland health.

If you're an undergraduate student interested in this work, reach out and we can make something happen.

Mapping Agroforestry Systems Across Sub-Saharan Africa

This project maps where different tree-crop agroforestry systems are practiced across Sub-Saharan Africa using spatial data and satellite imagery.

We are interested in understanding the trade-offs in tree-crop systems under extreme climate conditions.
